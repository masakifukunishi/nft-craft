# NFT Craft

## Summary
This site enables the minting and browsing of NFTs. It allows users to mint ERC-721 tokens and view them. Currently, the supported chains are the Testnets Sepolia and Mumbai.

## Tech Stack
### Contract
- Hardhat
- Solidity
- OpenZeppelin

### Node Provider
- Infura

### IPFS
- NFT.Storage

### Frontend / Backend
- TypeScript
- Next.js
- Tailwind CSS
- Redux
- Wagmi
- Moralis
- Auth.js
- siwe (Sign in with Ethereum)

### Supported Chains
#### Testnets
- Sepolia
- Mumbai
